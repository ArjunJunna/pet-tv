# Pet - TV
[Pet-TV](https://pet-tv-react.vercel.app/)  One stop video library App for pet lovers.

## Features
- Video Listing page
- Like/dislike a video
- Watchlater Management
- History Management
- Filter videos by category
- Search videos
- Playlist Management
- Authentication

## Pages
- Home page
- Explore page
- Playlist page
- Liked Videos page
- Watchlater page
- History page
- Single Video page
- Login, Signup page

## Tech Stack
- ReactJS
- React Router v6
- React Context API and useReducer
- React Player
- [Aqua UI](https://its-aqua-ui.netlify.app/) - CSS Component Library
- MockBee (For Backend)
- React Toastify
